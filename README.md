# Mundial de Alfajores

Proyecto final de e-commerce realizado para el curso de JavaScript.

La página permite visualizar productos de forma dinámica utilizando una API REST, agregar productos al carrito y mantener la compra guardada con `localStorage`.

## Tecnologías utilizadas

- HTML5 semántico
- CSS3
- Flexbox y Bootstrap
- JavaScript
- Fetch API
- LocalStorage
- Formspree

## Funcionalidades

- Productos cargados dinámicamente desde una API.
- Carrito de compras con contador, total y eliminación de productos.
- Persistencia del carrito al actualizar la página.
- Formulario de contacto.
- Diseño responsivo para distintos tamaños de pantalla.
- Publicación mediante GitHub Pages.
